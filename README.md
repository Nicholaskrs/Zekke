# Golang Project

## Setup project
 - Clone project
 - Run command
    ```
    go mod tidy
    ```

## Run project

 - Go to root folder
 - Run command
    ```
    go run main.go
    ```
- Server will run in `localhost:9000`


## Environment
- Copy environment from config.env
- Rename copied config.env into config.local.env

## Important Notes
- Currently all time are using time.now so it'll based on machine local time.
